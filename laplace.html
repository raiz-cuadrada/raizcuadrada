La transformada de Laplace es un procedimiento complejo para el motor matemático tradicional del sitio. Para garantizar resultados confiables, este recurso utiliza una API de Wolfram Alpha.

Una API (Application Programming Interface) es un conjunto de reglas que permite que dos programas se comuniquen entre sí, sin necesidad de conocer cómo está construido el otro.
