Visitas al sitio

Instalar la aplicación

Instala esta app en tu dispositivo para acceder más rápido y sin navegador.

Android (Chrome)

  1. Abre este sitio en Google Chrome
  2. Toca el menú ⋮ (arriba a la derecha)
  3. Selecciona Agregar a pantalla principal
  4. Selecciona "instalar" (Encontrarás la aplicacion en tu cajon de aplicaciones);

iPhone (Safari)

  1. Abre este sitio en Safari
  2. Toca el botón Compartir
  3. Selecciona Añadir a pantalla de inicio
  4. Confirma tocando Agregar

Computador (Chrome / Edge)

  1. Abre este sitio en Google Chrome o Microsoft Edge
  2. Toca el menú ⋮ (arriba a la derecha)
  3. Selecciona Enviar, guardar y compartir
  4. Instalar pagina como aplicación (o si deseas, como acceso directo)

Si no ves la opción de instalar, asegúrate de usar un navegador actualizado.
